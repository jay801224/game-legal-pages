Common Issues | 常見問題

The game doesn't load / shows a blank screen | 遊戲打不開或畫面空白

  • Make sure JavaScript is enabled in your browser. Inkwhisker requires JavaScript.
  • Try a different browser (Chrome / Firefox / Edge / Safari). Mobile Safari / Chrome Android also supported.
  • If you have an aggressive content blocker, allow scripts on the game's domain.
  • If you're behind a corporate firewall, some WebGL / Web Audio API features may be blocked.

確認瀏覽器有開啟 JavaScript；嘗試其他瀏覽器（Chrome / Firefox / Edge / Safari，行動版亦支援）；若有積極的內容封鎖工具，請開放本網域的腳本；公司防火牆可能會擋 WebGL / Web Audio API。

I lost my high scores | 我的分數不見了

  • Local Top-5 lives in your browser's localStorage. Clearing browser data, using Incognito mode, or switching browsers wipes them.
  • If you published a score to the cloud leaderboard, sign in with the same email to restore.

本機前 5 名分數儲存在瀏覽器的 localStorage。清除瀏覽器資料、無痕模式或換瀏覽器都會消失。若你曾上傳至雲端榜單，請以原 email 重新登入即可看到。

Audio doesn't play on iPhone | iPhone 上聽不到聲音

  • iOS Safari blocks audio until you tap the screen. The "Tap to Start" button at the menu unlocks the audio context.
  • If audio worked then went silent after switching tabs, return to the game tab and start a new round — iOS sometimes re-locks idle audio contexts.
  • Check the Silent Mode physical switch on the side of your iPhone.

iOS Safari 必須在使用者點擊後才會播放音效，請點按「Tap to Start」解鎖。若切換分頁後音效消失，回到遊戲分頁重新開始一局即可（iOS 有時會把閒置音效鎖回去）。也請確認 iPhone 側邊的靜音實體開關。

Touch controls aren't working on mobile | 手機觸控操作沒反應

  • The on-screen joystick + spell buttons appear automatically on touch devices. If they don't, force-reload the page (pull down to refresh on mobile).
  • Mobile orientation should be landscape for best experience, but portrait also works.

觸控裝置會自動顯示虛擬搖桿與咒法按鈕。若沒出現，請強制重新整理（手機可下拉頁面刷新）。橫向體驗最佳，直向也可玩。

Bug Reports | Bug 回報

If you found a bug — especially a reproducible one — please email with:

  • What you were doing when it happened.
  • Browser + OS version (e.g. Chrome 132 on Windows 11, Safari on iOS 18.5).
  • A screenshot or short video if you can.

發現 bug（特別是能重現的）請寄信附上：發生當下你在做什麼、瀏覽器與作業系統版本（例如 Windows 11 上的 Chrome 132、iOS 18.5 的 Safari）、若可附上截圖或短片。
